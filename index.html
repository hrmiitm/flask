Flask Setup And Installation

What is Flask?

Flask is a micro web framework written in Python. It is designed to make getting started quick and easy, with no CSS/JS framework.

  • Micro Web Framework
  • Fast, simple, and extensible to support database etc.
  • By: Armin Ronacher
  • Based On: Werkzeug WSGI and Jinja

Requirements

  1. Python Interpreter: We will use ANACONDA VENV
  2. Code Writer: We will use VS CODE

First Setup

  1. Open AnacondaNavigator --> Environment --> Create New (say hrmvenv)
  2. Open an Empty Folder (day1) in vs code and Create a File (app.py)
  3. Activate Virtual Environment (hrmvenv): Select Interpreter --> hrmvenv
  4. Open Terminal:Ctrl + ~
  5. Install Flask by runnning this command: pip install flask
  6. Run this command: python app.py after writing content inside app.py
#### app.py ```python from flask import Flask app = Flask(__name__) @app.route('/') def hello(): return 'Hello, World!' app.run() ```
